# Movie & Series Search Site

A mini Netflix-style interface to browse movies and series.  
Built with **HTML, CSS, and JavaScript**. Movie data is stored in a local **JSON file**.  
The site is **fully responsive** and includes a **search bar** to quickly find movies or series.

## Features
- Search movies & series using the search bar
- Multi-page layout
- Local JSON database
- Fully responsive design for all screen sizes
